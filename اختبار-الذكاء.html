اختبار الذكاء المتقدم

اختبر قدراتك العقلية مع مجموعة متنوعة من الأسئلة المصممة لقياس ذكائك

📋 طرق تشغيل التطبيق:

  1. الطريقة الأولى (مستحسنة): تشغيل ملف run-iq-test.bat أو run-iq-test.ps1
  2. الطريقة الثانية: فتح مجلد dist وتشغيل ملف index.html
  3. الطريقة الثالثة: استخدام الأوامر التالية في Terminal:
npm install
npm run build
npm run preview
⚠️ تحذير: يتطلب تشغيل التطبيق وجود Node.js على النظام

🎯 مميزات التطبيق:

  • ✅ 15 سؤال متنوع في مجالات مختلفة
  • ✅ نظام توقيت ذكي مع مكافآت السرعة
  • ✅ تحليل مفصل للأداء والنتائج
  • ✅ تصنيف مستوى الذكاء
  • ✅ واجهة عربية حديثة ومتجاوبة
  • ✅ إمكانية المشاركة على وسائل التواصل
